# Django Polls Tutorial
B00159780

## How to run:
1. Create virtual environment: `python -m venv django_env`
2. Activate: `django_env\Scripts\activate` (Windows) or `source django_env/bin/activate` (Mac/Linux)
3. Install dependencies: `pip install -r requirements.txt`
4. Run server: `python manage.py runserver`
5. Visit: http://127.0.0.1:8000/polls/
6. Admin: http://127.0.0.1:8000/admin/ (login with username: temic, password: admin12345)

## Features:
- Polls app with Questions and Choices
- Voting functionality
- Admin interface
- SQLite database included with sample data
